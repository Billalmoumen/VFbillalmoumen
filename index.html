Veille technologique : Le Cloud Computing

Le Cloud Computing désigne la fourniture de services informatiques à la demande via Internet : serveurs, stockage, bases de données, réseaux, logiciels, etc. Ces services permettent aux entreprises de ne plus avoir à gérer physiquement des infrastructures informatiques locales, et d’accéder à des ressources flexibles en payant uniquement ce qu’elles consomment.

Il existe trois modèles principaux :

  • IaaS (Infrastructure as a Service) : Location d'infrastructures informatiques comme Amazon AWS ou Microsoft Azure.
  • PaaS (Platform as a Service) : Mise à disposition de plateformes de développement prêtes à l’emploi (ex : Google App Engine).
  • SaaS (Software as a Service) : Utilisation de logiciels directement via le web sans installation (ex : Gmail, Dropbox).
